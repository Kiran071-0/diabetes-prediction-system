# 🩺 Diabetes Prediction System Using Machine Learning

##  Overview
This project is a machine learning-based web application that predicts whether a patient is diabetic or non-diabetic based on medical attributes. It also compares multiple machine learning models to identify the best-performing algorithm.

## Objective
- Predict diabetes using patient health data
- Compare multiple ML classification models
- Provide real-time user prediction through a web interface

##  Dataset
Pima Indians Diabetes Dataset  
Features include:
- Pregnancies, Glucose, Blood Pressure  
- Skin Thickness, Insulin, BMI  
- Diabetes Pedigree Function, Age  
- Outcome (0 = Non-Diabetic, 1 = Diabetic

##  Machine Learning Models
- Logistic Regression  
- Decision Tree  
- Random Forest  
- K-Nearest Neighbors (KNN)  
- Support Vector Machine (SVM)

## Tech Stack
- Python  
- Pandas & NumPy  
- Scikit-learn  
- Streamlit  

## Features
- User input-based prediction system  
- Real-time diabetes detection  
- Model performance comparison dashboard  
- Interactive and simple UI

pip install -r requirements.txt

## Result

The system predicts diabetes risk and identifies the most accurate machine learning model for medical diagnosis.
